# Weekly Indices Review — Decision Framework

## Purpose
Define what the Weekly Indices Review is trying to decide each cycle.

## Primary decisions
- What is the current macro regime?
- What is the current geopolitical regime?
- Which stock-index exposures best fit the current 3–12 month regime?
- Which current exposures still deserve capital?
- Which exposures are replaceable?
- Which new exposures deserve entry into the live opportunity board?
- What should be added, held, reduced, or closed?

## Core principles
- Capital must earn its place.
- Opportunity cost matters.
- Region / style / size leadership matters.
- Second-order effects matter.
- Cash is an active choice.
- The report must remain compact and decision-useful.

## Exposure model
The system should think in terms of **index exposures**, not ETF sleeves and not options structures.

Each exposure should map:
- benchmark index
- primary proxy
- alternative proxy
- role in portfolio
- thesis
- timing state
