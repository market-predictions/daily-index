# FX Deep Research Runtime — Prep-First ChatGPT Production Flow

This file is the dedicated **prep-first ChatGPT runtime** for the Weekly FX Review.

Use this file when you want the canonical ChatGPT-first production path with an explicit mandatory state refresh before research begins.

---

## Core operating rule

This runtime must **not** start immediately with deep research.

Before doing any research or writing any report, you must treat the technical/state refresh as a mandatory prerequisite.

The required sequence is:

1. confirm that `.github/workflows/refresh-fx-state.yml` has been run successfully
2. confirm that the refreshed state commit landed on `main`
3. only then begin the deep-research run
4. generate the Weekly FX Review in markdown
5. write the markdown file to `output_split_test/weekly_fx_review_YYMMDD.md`
6. leave promote / render / email to the downstream GitHub workflow

If the prep step was not run first, the ChatGPT run should be treated as blocked by stale state.

## Fresh research requirement

Once prep is complete, begin a **fresh deep-research pass**.

At minimum, refresh:

### A. Global macro regime
- growth backdrop
- inflation backdrop
- labor-market direction where relevant
- risk-on / risk-off regime
- energy / commodity backdrop
- trade-policy backdrop
- major market stressors and second-order effects

### B. Central banks and policy divergence
Refresh the current stance / latest relevant decision / reaction-function framing for:
- Fed
- ECB
- BoE
- BoJ
- SNB
- BoC
- RBA
- RBNZ
- Banxico when MXN is in scope

### C. Geopolitics and event risk
Refresh the active geopolitical and event-risk layer relevant to FX allocation, including:
- war / conflict escalation or de-escalation
- energy-shock implications
- sanctions / tariff / trade-policy developments
- other cross-border macro stressors that matter for FX regime selection

### D. Current market-validating facts
Refresh only the facts needed to support strategic judgment.
Do not turn the report into a random news digest.
Research must serve portfolio judgment.
