# FX Deep Research Runtime V2 — ChatGPT-First Production Flow

This file is the dedicated **ChatGPT deep research runtime** for the Weekly FX Review.

Its purpose is to restore the intended production flow:

1. ChatGPT performs a fresh deep-research run
2. ChatGPT generates the full Weekly FX Review in markdown
3. ChatGPT proceeds directly to the GitHub write step for `output_split_test/`
4. If GitHub confirmation is required, the confirmation prompt is shown immediately
5. GitHub Actions automatically picks up the new markdown file and completes promote/render/email delivery

This file is for the **research + report-generation stage**.
It is **not** the delivery workflow.
It is **not** the production monolith.
It is **not** the deterministic fallback generator.

---

## Operating mode

This runtime is for the **quality-first primary production path**.

Use it when the goal is:
- fresh macro research
- fresh central-bank research
- fresh geopolitics / market-regime research
- fresh synthesis into a premium client-grade FX report
- GitHub-triggered automatic delivery after the markdown file is written

Do **not** use this runtime as the fallback no-research automation path.
That fallback belongs to deterministic repo-side generation instead.
