# Project Bootstrap — Daily Index OS

Use this repository as the live operating source of truth for the AEX weekly options system.

## Read order for serious work
1. `control/AEX_SYSTEM_INDEX.md`
2. `control/AEX_CURRENT_STATE.md`
3. `control/AEX_NEXT_ACTIONS.md`
4. only then the minimum relevant execution file(s)

## System purpose
Build and maintain a disciplined operating system for:
- macro / geopolitical / Europe-sensitive AEX weekly options analysis
- dedicated AEX technical confirmation
- options-surface regime assessment
- defined-risk financed-convexity structures
- explicit state and risk tracking
- report + machine trade plan output
- conservative automation with human approval first

## Guardrails
- Technicals are confirmation, not the whole decision engine.
- Do not collapse the four layers into one monolith.
- No-trade is the default unless the burden of proof is met.
- Only approved financing families are allowed.
- Do not claim delivery succeeded without a manifest or receipt.
