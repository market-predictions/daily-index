# Templates

These JSON templates define the expected machine-facing artifacts for the AEX options system.

## Core templates
- `aex_macro_snapshot.template.json`
- `aex_primary_technical_snapshot.template.json`
- `aex_cross_market_confirmation.template.json`
- `aex_option_surface_snapshot.template.json`
- `aex_option_portfolio_state.template.json`
- `aex_option_risk_state.template.json`
- `aex_option_trade_plan.template.json`

## Rule
Treat these as shape references, not as live state.
